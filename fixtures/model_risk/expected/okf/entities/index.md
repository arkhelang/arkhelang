# Entities

Entities declared in the model_risk module.

- [ModelChange](ModelChange.md): A proposed change to a model under change control.
- [TradingDesk](TradingDesk.md): A front-office desk that owns and uses models.
- [ValidationReview](ValidationReview.md): An independent validation exercise against one model.
