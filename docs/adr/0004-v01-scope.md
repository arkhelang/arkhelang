# ADR 0004: v0.1 ships four things

Date: 2026-07-17
Status: Accepted

## Context

The space is converging fast: OSI/Apache Ossie, Google's OKF, Microsoft's
Agent Control Specification, and at least one open near-neighbour all
appeared within two quarters. Specs die of ambition more often than of
competition, and this one's influence window is set by other people's pace.
The scope pressure is real: four core emitters, a flagship module, and
evidence references were all sketched before a line of code existed.

## Decision

v0.1 ships exactly four things:

1. The specification: metamodel as JSON Schema, prose, and golden-file
   fixtures (fixture one: a financial-services model-risk domain; fixture
   two: a mining maintenance domain).
2. A validator (`arkhe validate`).
3. The tool-contract IR (`arkhe contracts`).
4. One emitter: native library stubs with guard pre-checks and structured
   refusals.

Everything else (Cedar, JSON Schema models, OKF bundles, protocol emitters,
the AI-estate flagship module, OWL/RDF export) is explicitly deferred to
v0.2 or later.

## Consequences

- Weeks, not quarters. The golden files are written by hand before the
  validator exists; the validator is written to make them pass.
- The release announcement is the guard-refusal demo: an agent attempts an
  action, the guard refuses with a legible reason.
- Scope additions to v0.1 require superseding this ADR, which is intended to
  be embarrassing to do.
